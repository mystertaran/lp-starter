## Design Principles

1. **Typography carries the design** ‚Äî before adding a graphic, see if a heavier weight and a blank line will do the job. The page should work in black and white.
2. **One decision per screen** ‚Äî every section has exactly one next action for the reader. If two CTAs compete, one is wrong.
3. **Receipts over adjectives** ‚Äî numbers, integrations, channels. {{BANNED_COPY_WORDS ‚Äî e.g. 'No "innowacyjny", no "rewolucyjny", no "lider"'}}.
4. **Accent is rare and earned** ‚Äî used to mark conversion moments and nothing else. If it appears everywhere, it means nothing.
