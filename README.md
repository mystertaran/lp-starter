# fabos-lp

Landing page dla FabOS — platformy integrującej kanały sprzedaży e-commerce (magazyn, zamówienia, wysyłki, zwroty, księgowość) w jedno miejsce.

## Stack

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- Motion (framer-motion) dla animacji
- Design tokens: Impeccable + SkillUI (anti-AI-slop)

## Development

```bash
pnpm install
pnpm dev
```
